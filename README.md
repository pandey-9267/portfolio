## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/pandey-9267/portfolio.git
```

### Frontend

```bash
cd portfolio-frontend
npm install
npm run dev
```

### Backend

```bash
cd portfolio-backend
npm install
npm run dev
```

---

## 🔐 Environment Variables

Create a `.env` file inside `portfolio-backend`.

```env
PORT=5000

MONGO_URI=YOUR_MONGODB_CONNECTION_STRING
```
